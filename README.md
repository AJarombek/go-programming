# go-programming

### Overview

Go programming code, tested and run on Docker containers.

### Commands

**Fix the `go.mod` file.**

```bash
go mod tidy
```

**Format the Go code.**

```bash
go fmt
```

### Directories

| Directory Name     | Description                                                                                             |
|--------------------|---------------------------------------------------------------------------------------------------------|
| `composite-types`  | Go code dealing with the basics of composite types, based on Chapter 4 of The Go Programming Language.  |
